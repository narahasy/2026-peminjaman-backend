# 2026-peminjaman-backend

Proyek REST API untuk sistem peminjaman menggunakan **ASP.NET Core 8** dan **Entity Framework Core**.

## Teknologi yang Digunakan
* **Framework:** .NET 8 (Web API)
* **Database:** SQLite
* **ORM:** Entity Framework Core
* **Tools:** Swagger UI untuk dokumentasi API

## Cara Menjalankan
1. Masuk ke folder proyek: `cd 2026-peminjaman-backend`
2. Jalankan perintah: `dotnet run`
3. Akses API melalui: `http://localhost:5179/swagger`

## Fitur
* **CRUD Customers:** Pengelolaan data peminjam.
* **Database Seeding:** Otomatis mengisi data Admin, Budi, dan Siti saat pertama kali dijalankan.
